The Problem...

The skills and responsibilities of Front-End developers are becoming ambiguous.

The old journey ...

  • Start with CSS and HTML
  • Learn some JavaScript, mostly jQuery
  • Back End vs Front End

The new journey ...

  • Start with CSS and HTML
  • Don't bother too much ...
  • JavaScript is everywhere
  • Bingo! - I am a Full Stack Developer

The Results


Companies are having trouble finding Developers proficient in HTML & CSS

People are steering away because its not as impressive and profitable

Definitions

Front-End Engineer
Front-End (maybe middleware) functionality developer
Front-End Designer
Front-end visual/interaction developer
Front-End Full stack Developer
Fully competent in both of the above.
